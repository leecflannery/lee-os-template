# Skill: Paper Summary

**Trigger:** "Summarize this paper" (with PDF or link)

## Instructions

1. Read/fetch the paper
2. Generate a summary following the template
3. Save to `/research/[topic]/[paper-name].md`

## Summary Template

```markdown
# [Paper Title]

**Authors:** [Names]
**Published:** [Date, venue]
**Link:** [URL]

## One-Sentence Summary
[The core contribution in plain language]

## Key Contributions
1. [Contribution 1]
2. [Contribution 2]
3. [Contribution 3]

## Method
[How they did it, simplified]

## Results
[Key findings, numbers if relevant]

## Limitations
[What the paper doesn't address]

## Relevance
[Why this matters for your interests and current work]

## Discussion Questions
1. [Question]
2. [Question]

## Tags
[topic tags for organization]
```

## Behavior

- Write for a mixed audience (researchers + non-experts)
- Avoid jargon or explain it
- Focus on "so what" -- why does this matter?
