# Event Checklist

Full lifecycle checklist for events. Adapt per event type -- not every item applies every time.

## Trigger Phrases

- `/event-checklist`
- "Set up a checklist for [event]"
- "What do I need to do for [event]?"

## How to Use

When triggered, create a copy of the relevant sections in the event file or in `/tasks/`. Check off items as they're completed. Skip items that don't apply.

---

## Phase 1: Pre-Event (2-4 weeks out)

### Venue
- [ ] Book room -- confirm date, time, capacity
- [ ] Confirm AV setup (projector, HDMI, sound system)
- [ ] Confirm entrance instructions for attendees
- [ ] Note seating layout (depends on format)

### Registration
- [ ] Create event page (Luma, Eventbrite, or equivalent)
- [ ] Write event description and title
- [ ] Configure registration questions
- [ ] Write confirmation email
- [ ] Set capacity cap if needed

### Speaker / Co-host Coordination
- [ ] Confirm speaker or co-host -- topic, title, date
- [ ] Get permission to record (audio + video)
- [ ] Coordinate presentation content -- review draft, align on timing
- [ ] Confirm speaker arrival time

### Recording / Filming
- [ ] Book videographer (if applicable)
- [ ] Source audio recording solution
- [ ] Prepare interview questions if filming attendee interviews

### Content Prep
- [ ] Create presentation / slides (if applicable)
- [ ] Prepare discussion questions or breakout prompts
- [ ] Write run of show with minute-by-minute timeline
- [ ] Assign pre-read if applicable

### Promotion (start 2 weeks out)
- [ ] Email list
- [ ] Community channels (Slack, Discord, etc.)
- [ ] Social media posts
- [ ] Cross-promote at other events
- [ ] Ask speaker to share with their network
- [ ] Monitor registration count -- adjust promotion if under target

### Materials & Logistics
- [ ] Laptop + adapters for projector
- [ ] Backup: slides on USB drive or second device
- [ ] Phone charger
- [ ] Signage or QR codes (if applicable)

## Phase 2: Day-Of Setup

### Before Doors Open (arrive 40-60 min early)

| Time | Task |
|------|------|
| T-60 min | Arrive, access space |
| T-55 min | Set up projector, test AV |
| T-50 min | Arrange seating for format |
| T-45 min | AV check -- confirm font legibility from back of room |
| T-45 min | Put out signage, printed materials |
| T-40 min | Camera/recording setup (if filming) |
| T-30 min | Put out snacks (if applicable) |
| T-10 min | Final tech check |
| T-0 | Doors open |

### Checks Before Go
- [ ] Slides loaded and displaying correctly
- [ ] Audio recording running
- [ ] Video recording started (if applicable)
- [ ] Phone on silent
- [ ] Know the wifi password for attendees

---

## Phase 3: During Event

### Opening (first 10 min)
- [ ] Welcome -- introduce yourself, co-host, and the event
- [ ] Frame the session (what this is, what to expect, how long)
- [ ] If recording: announce it, ask for verbal consent
- [ ] Point out logistics (snacks, bathrooms, QR codes)

### Running the Event
- [ ] Stick to run of show timing (or consciously deviate)
- [ ] Watch energy -- if a section is running long and energy drops, cut it
- [ ] If breakouts: float between groups
- [ ] Capture standout moments for post-event content

### Content Capture (during event)
- [ ] Confirm audio recording is still running (check at midpoint)
- [ ] Photos -- grab 3-5 candid shots during high-energy moments
- [ ] Note quotable things people say
- [ ] Note anyone who stands out (potential speaker, collaborator)

### Closing
- [ ] Reconvene -- one takeaway per person or group
- [ ] Promote next event
- [ ] Thank co-hosts, speakers, venue
- [ ] No hard stop -- let people linger for networking

## Phase 4: Post-Event

**Run `/post-event` for the full debrief and ops checklist.**

Quick reference of the 24-hour ops checklist:

### Comms (day of / next morning)
- [ ] Thank you via event platform
- [ ] Thank co-hosts/speakers personally

### Capture (within 24 hours)
- [ ] Process transcript (if recorded)
- [ ] Upload photos/video
- [ ] Export attendee data

### Content (within 48 hours)
- [ ] Draft social media recap (run `/voice-critic` before posting)
- [ ] Identify clips, quotes, or takeaways for social
- [ ] Schedule video editing if filmed

### Network (within 48 hours)
- [ ] Mine attendee list for potential collaborators
- [ ] Connect with notable attendees
- [ ] Follow up on specific interest expressed

### System (within 24 hours)
- [ ] Update event file with post-event data
- [ ] Create/update people files for new contacts
- [ ] Create follow-up tasks
